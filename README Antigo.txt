<h1 align="center"> Decodificador </h1>
<p>Desafio dado pela Alura de criar um decodificador de mensagens (criptografar e descriptografar)</p>
<p><strong>Recursos usados:</strong></p>
<ul>
  <li>HTML e CSS (estilização e posicionamentos)</li>
  <li>Java Script usado no VS Code</li>
</ul>

<p><strong>Requisitos e condições do desafio:</strong></p>
<ul>
  <li>Trocar letras específicas por palavras. Ex: A letra "e" é convertida para "enter"
A letra "i" é convertida para "imes"
A letra "a" é convertida para "ai"
A letra "o" é convertida para "ober"
A letra "u" é convertida para "ufat"</li>
  <li>Funcionar apenas com letras minúsculas e sem acentos</li>
</ul>
